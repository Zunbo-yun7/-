# Android App 构建说明

## 方式一：Android Studio（推荐）

### 步骤

1. **下载安装 Android Studio**
   - 官网：https://developer.android.com/studio
   - 安装时确保勾选「Android SDK」

2. **导入项目**
   - 打开 Android Studio
   - 选择 File → Open
   - 选择 `android_app` 文件夹
   - 等待 Gradle 同步完成（约 3-5 分钟）

3. **生成签名密钥（发布版本需要）**
   - Android Studio → Build → Generate Signed Bundle / APK
   - 选择 APK → Create New
   - 填写密钥库信息，保存密钥

4. **构建 Debug APK**
   - Android Studio → Build → Build Bundle(s) / APK(s) → Build APK(s)
   - 完成后右下角会显示 "APK generated successfully"
   - APK 位置：`app/build/outputs/apk/debug/app-debug.apk`

5. **构建 Release APK**
   - Android Studio → Build → Generate Signed Bundle / APK(s)
   - 选择 APK → 选择之前创建的密钥
   - 构建完成，APK 即为可发布的安装包

## 方式二：命令行构建

### 环境要求

- JDK 17+
- Android SDK (设置 `ANDROID_HOME` 环境变量)
- Gradle 8.2（Android Studio 自带）

### 命令

```bash
cd android_app

# 构建 Debug APK
./gradlew assembleDebug

# 构建 Release APK（需配置签名）
./gradlew assembleRelease
```

APK 输出位置：`android_app/app/build/outputs/apk/`

---

## 注意事项

- **本 App 需要配合电脑端词典使用**：手机 App 通过 WiFi 访问电脑上的 Flask 服务器
- 确保手机和电脑在同一局域网内
- 构建 Release APK 前需要配置签名密钥

---

## WiFi 连接设置

1. 在电脑上运行 `run.bat`，确认服务启动成功
2. 查看电脑的局域网 IP 地址（例如 `192.168.1.100`）
3. 修改 `LocalServerActivity.java` 中的 `LOCAL_URL`：
   ```java
   private static final String LOCAL_URL = "http://192.168.1.100:5000";
   ```
4. 重新构建 APK 安装到手机
5. 手机和电脑必须处于同一 WiFi 网络
